# Moooscle Press Kit

This press kit provides official visual assets, colors, and branding elements  
for anyone who wants to create memes, posts, banners, or community content  
related to Moooscle (MSCL).

Moooscle is a community‑driven, identity‑focused project.  
All assets are free to use for non‑commercial, community purposes.
